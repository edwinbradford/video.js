AV Neue Theme.

The 'main' branch is an untouched mirror of the videojs/video.js 'main' branch used only for testing. The AV Neue theme development is in the 'themes/avneue' branch.

On WSL Ubuntu the development environment for Video.js 7.12.1 uses Node LTS 14.16.1. Refer to the video.js contributing.md guide for installation after which run npm update if either npm test or npm build do not run as expected.
